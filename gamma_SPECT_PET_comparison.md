# γ 相机、SPECT、PET 成像原理对比

## 一、三者的基本关系

γ 相机、SPECT 和 PET 都属于**核医学成像**，共同特点是：

> 将放射性示踪剂引入人体，通过探测其发出的射线，间接反映器官或组织的功能、代谢、血流、受体分布等信息。

三者的区别主要在于：

| 项目 | γ 相机 | SPECT | PET |
|---|---|---|---|
| 成像类型 | 平面显像 | 单光子发射断层显像 | 正电子发射断层显像 |
| 是否断层成像 | 一般不是 | 是 | 是 |
| 探测对象 | 单个 γ 光子 | 单个 γ 光子 | 一对 511 keV 湮灭光子 |
| 是否需要准直器 | 需要 | 需要 | 通常不需要机械准直器 |
| 典型空间信息 | 二维投影 | 三维断层 | 三维断层 |
| 定量能力 | 较弱 | 中等 | 较强 |

需要注意的是：

> **SPECT 本质上是在 γ 相机基础上发展出来的断层成像方法。**  
> γ 相机获得的是某一方向上的二维投影图像，而 SPECT 通过 γ 相机围绕人体旋转采集多个角度的投影，再重建出断层图像。

## 二、γ 相机

### 1. 成像原理

γ 相机又称**闪烁照相机**，主要用于探测体内放射性核素发出的 γ 光子。

基本过程为：

1. 向人体注入、口服或吸入放射性示踪剂；
2. 示踪剂在特定器官或组织中分布；
3. 放射性核素发出 γ 光子；
4. γ 光子经过**准直器**筛选方向；
5. 光子进入闪烁晶体，产生可见光；
6. 光电倍增管或半导体探测器将光信号转换为电信号；
7. 根据光信号的位置和强度形成二维图像。

简化表示为：

```text
体内放射性示踪剂
        ↓ 发射 γ 光子
准直器筛选方向
        ↓
闪烁晶体产生光信号
        ↓
光电转换与定位
        ↓
二维平面图像
```

γ 相机的关键部件是：

| 部件 | 作用 |
|---|---|
| 准直器 | 只允许特定方向的 γ 光子进入探测器 |
| 闪烁晶体 | 将 γ 光子转换成可见光 |
| 光电倍增管 / 光电探测器 | 将光信号转换成电信号 |
| 位置计算电路 | 判断光子入射位置 |
| 能量窗电路 | 筛选符合能量范围的光子，减少散射干扰 |

### 2. 使用的示踪剂

γ 相机常用的是**发射 γ 光子的单光子核素**。

常见核素包括：

| 放射性核素 | 常见用途 |
|---|---|
| Tc-99m | 骨显像、肾显像、心肌灌注、肝胆显像等 |
| I-123 | 甲状腺显像、神经内分泌显像 |
| I-131 | 甲状腺疾病诊断和治疗 |
| Tl-201 | 心肌显像 |
| Ga-67 | 炎症、肿瘤显像 |
| In-111 | 白细胞显像、受体显像 |

其中 **Tc-99m** 是 γ 相机和 SPECT 中最常用的核素之一，因为它 γ 光子能量适中，半衰期合适，成像质量较好，患者接受剂量相对较低。

### 3. 衰变类型

γ 相机使用的核素主要通过以下方式产生 γ 光子：

| 类型 | 说明 |
|---|---|
| 同质异能跃迁 | 激发态原子核退激发，发出 γ 光子，例如 Tc-99m |
| 电子俘获 | 原子核俘获内层电子，常伴随 γ 射线或特征 X 射线发射，例如 I-123、Tl-201 |
| β⁻ 衰变伴随 γ 发射 | 部分核素 β⁻ 衰变后子核处于激发态，再发射 γ 光子，例如 I-131 |

γ 相机关注的是：

> 放射性核素直接发出的单个 γ 光子。

### 4. 成像性能

γ 相机的特点是：

| 性能指标 | 特点 |
|---|---|
| 空间分辨率 | 中等，受准直器、晶体厚度、光子能量影响明显 |
| 灵敏度 | 受准直器限制，通常较低 |
| 成像速度 | 平面显像较快 |
| 定量能力 | 较弱，主要用于定性或半定量分析 |
| 图像维度 | 多为二维投影图像 |
| 解剖定位能力 | 单独使用时较弱 |

γ 相机的一个重要限制是：

> 准直器虽然可以确定光子方向，但会挡掉大量光子，因此提高空间分辨率通常会牺牲灵敏度。

### 5. 适用范围

γ 相机适合进行**平面功能显像**，常用于：

| 应用 | 说明 |
|---|---|
| 甲状腺显像 | 观察甲状腺位置、形态、摄取功能 |
| 骨显像 | 筛查骨转移、骨损伤、炎症等 |
| 肾动态显像 | 评价肾血流、肾功能、尿路通畅性 |
| 肝胆显像 | 评价胆道通畅情况 |
| 心肌灌注平面显像 | 评价心肌血流分布 |
| 全身显像 | 如全身骨显像、某些肿瘤或炎症筛查 |

总结：

> γ 相机适合观察放射性示踪剂在体内的整体分布，结构简单、应用广泛，但断层定位和定量能力有限。

## 三、SPECT

### 1. 成像原理

SPECT 是 **Single Photon Emission Computed Tomography**，即**单光子发射计算机断层成像**。

它的基本原理是：

1. 体内示踪剂发出单个 γ 光子；
2. γ 相机围绕人体旋转；
3. 从多个角度采集二维投影；
4. 计算机对投影数据进行断层重建；
5. 得到横断面、冠状面、矢状面或三维图像。

简化表示为：

```text
体内单光子示踪剂发射 γ 光子
        ↓
旋转 γ 相机多角度采集
        ↓
获得多个方向的投影数据
        ↓
计算机断层重建
        ↓
三维功能图像
```

SPECT 与普通 γ 相机的核心区别是：

> γ 相机是单角度或少角度二维显像，SPECT 是多角度采集后进行断层重建。

### 2. 使用的示踪剂

SPECT 使用的示踪剂与 γ 相机高度相似，主要是**单光子发射核素**。

常见核素包括：

| 放射性核素 | 常见示踪剂或用途 |
|---|---|
| Tc-99m | 骨显像、心肌灌注、脑血流、肾显像 |
| I-123 | 甲状腺、DAT 显像、MIBG 显像 |
| I-131 | 甲状腺癌、神经内分泌肿瘤诊疗 |
| In-111 | 白细胞显像、奥曲肽显像 |
| Ga-67 | 炎症、淋巴瘤等 |
| Tl-201 | 心肌灌注、肿瘤显像 |

典型 SPECT 示踪剂包括：

| 示踪剂 | 主要用途 |
|---|---|
| Tc-99m-MDP | 骨显像 |
| Tc-99m-MIBI | 心肌灌注、甲状旁腺显像 |
| Tc-99m-HMPAO / ECD | 脑血流显像 |
| I-123-MIBG | 心脏交感神经、神经内分泌肿瘤 |
| I-123-FP-CIT | 多巴胺转运体显像 |
| In-111-octreotide | 生长抑素受体显像 |

### 3. 衰变类型

SPECT 使用的是**单光子发射核素**，其衰变类型与 γ 相机类似。

主要包括：

| 衰变类型 | 说明 |
|---|---|
| 同质异能跃迁 | 如 Tc-99m 发射 140 keV γ 光子 |
| 电子俘获 | 如 I-123、Tl-201 |
| β⁻ 衰变伴 γ 发射 | 如 I-131 |

SPECT 探测的是：

> 单个 γ 光子。

这也是它被称为“单光子发射断层成像”的原因。

### 4. 成像性能

SPECT 相比普通 γ 相机的优势是：

| 性能指标 | 特点 |
|---|---|
| 空间定位 | 明显优于平面 γ 相机 |
| 图像维度 | 可获得三维断层图像 |
| 对病灶重叠的分辨能力 | 优于平面显像 |
| 灵敏度 | 仍受准直器限制 |
| 空间分辨率 | 一般低于 PET |
| 定量能力 | 优于普通平面显像，但通常弱于 PET |
| 扫描时间 | 通常比平面显像更长 |

SPECT 的主要限制仍然来自准直器：

> 准直器会丢弃大量非指定方向的 γ 光子，因此 SPECT 的探测灵敏度不高。

但 SPECT 可以与 CT 结合形成 **SPECT/CT**，从而改善：

| 改善方面 | 说明 |
|---|---|
| 解剖定位 | CT 提供结构图像 |
| 衰减校正 | 提高定量准确性 |
| 病灶定位 | 更容易判断病灶所在器官或组织 |

### 5. 适用范围

SPECT 适合需要**断层定位的功能显像**，常见应用包括：

| 应用 | 说明 |
|---|---|
| 心肌灌注显像 | 诊断冠心病、心肌缺血、心肌梗死 |
| 骨 SPECT | 提高骨病灶定位能力 |
| 脑血流显像 | 评价脑灌注异常 |
| 帕金森病相关显像 | 多巴胺转运体显像 |
| 甲状旁腺显像 | 定位甲状旁腺腺瘤 |
| 肿瘤及炎症显像 | 如神经内分泌肿瘤、感染灶定位 |
| 肾功能断层显像 | 某些肾脏功能评估 |

总结：

> SPECT 适合在 γ 相机平面显像基础上进一步获得断层定位信息，尤其常用于心肌、骨、脑、内分泌和炎症显像。

## 四、PET

### 1. 成像原理

PET 是 **Positron Emission Tomography**，即**正电子发射断层成像**。

其核心原理与 γ 相机和 SPECT 不同。

PET 示踪剂中的核素发生 **β⁺ 衰变**，释放正电子。正电子在组织中运动一小段距离后，与电子相遇并发生湮灭，产生两个方向近似相反、能量均为 **511 keV** 的 γ 光子。

PET 探测器环绕人体，同时探测这两个光子，利用**符合探测**确定湮灭事件所在的响应线。

过程如下：

```text
PET 示踪剂发生 β⁺ 衰变
        ↓
释放正电子
        ↓
正电子与电子湮灭
        ↓
产生两个方向近似相反的 511 keV 光子
        ↓
环形探测器进行符合探测
        ↓
计算机断层重建
        ↓
三维代谢或分子功能图像
```

PET 的关键特征是：

> 它不是直接探测一个 γ 光子，而是探测一对同时产生、方向相反的 511 keV 湮灭光子。

### 2. 使用的示踪剂

PET 使用的是**正电子发射核素**标记的示踪剂。

常见核素包括：

| 放射性核素 | 半衰期特点 | 常见用途 |
|---|---|---|
| F-18 | 半衰期较适中，应用最广 | 肿瘤、脑、心脏显像 |
| C-11 | 半衰期短 | 神经递质、药物代谢研究 |
| N-13 | 半衰期短 | 心肌灌注 |
| O-15 | 半衰期很短 | 血流、氧代谢研究 |
| Ga-68 | 发生器来源 | 受体显像、前列腺癌显像 |
| Cu-64 | 半衰期较长 | 肿瘤、免疫显像 |
| Zr-89 | 半衰期较长 | 抗体 PET 显像 |

典型 PET 示踪剂包括：

| 示踪剂 | 主要用途 |
|---|---|
| F-18-FDG | 葡萄糖代谢显像，肿瘤、脑、心脏 |
| F-18-NaF | 骨代谢显像 |
| F-18-FDOPA | 多巴胺代谢、神经内分泌肿瘤 |
| C-11-methionine | 脑肿瘤、氨基酸代谢 |
| N-13-ammonia | 心肌灌注 |
| O-15-water | 血流灌注研究 |
| Ga-68-DOTATATE | 神经内分泌肿瘤受体显像 |
| Ga-68-PSMA | 前列腺癌显像 |
| F-18-PSMA | 前列腺癌显像 |

### 3. 衰变类型

PET 使用的核素主要发生：

> **β⁺ 衰变，也就是正电子衰变。**

一般形式为：

```text
母核 → 子核 + 正电子 + 中微子
```

正电子产生后，与电子发生湮灭：

```text
正电子 + 电子 → 2 个 511 keV γ 光子
```

因此 PET 实际探测的是：

> 正电子湮灭后产生的一对 511 keV γ 光子。

### 4. 成像性能

PET 的成像性能通常优于 SPECT，尤其是在灵敏度和定量能力方面。

| 性能指标 | PET 特点 |
|---|---|
| 空间分辨率 | 通常优于 SPECT |
| 灵敏度 | 高于 SPECT，因为不依赖机械准直器 |
| 定量能力 | 较强，可进行 SUV 等半定量或定量分析 |
| 断层能力 | 天然适合三维断层成像 |
| 分子功能信息 | 很强，适合代谢、受体、酶活性、转运体显像 |
| 成像速度 | 通常较快 |
| 成本 | 较高 |
| 对设备要求 | 高，需要符合探测和复杂重建 |

PET 的高灵敏度来自于：

> PET 通过两个 511 keV 光子的符合探测确定响应线，不需要像 SPECT 那样依赖机械准直器筛选方向。

不过 PET 也有一些限制：

| 限制 | 说明 |
|---|---|
| 正电子射程 | 正电子湮灭前会移动一小段距离，影响空间分辨率 |
| 非共线性 | 两个 511 keV 光子并非完全 180°，影响定位精度 |
| 示踪剂成本高 | 某些核素需要回旋加速器生产 |
| 半衰期短 | C-11、N-13、O-15 等需要近距离制备和使用 |
| 设备昂贵 | PET/CT、PET/MR 成本较高 |

PET 常与 CT 或 MRI 结合：

| 设备 | 优点 |
|---|---|
| PET/CT | 兼具代谢信息和解剖定位，临床最常见 |
| PET/MR | 软组织分辨率好，适合神经系统、盆腔、儿童等场景 |

### 5. 适用范围

PET 适合需要高灵敏度、高定量能力和分子代谢信息的场景。

常见应用包括：

| 应用 | 说明 |
|---|---|
| 肿瘤诊断与分期 | F-18-FDG PET/CT 用于多种恶性肿瘤 |
| 疗效评价 | 观察治疗前后代谢活性变化 |
| 肿瘤复发监测 | 区分瘢痕、坏死与活性肿瘤 |
| 脑功能与代谢显像 | 阿尔茨海默病、癫痫灶定位、脑肿瘤 |
| 心肌代谢与灌注 | 心肌活性、心肌缺血评价 |
| 前列腺癌显像 | PSMA PET |
| 神经内分泌肿瘤 | DOTATATE PET |
| 炎症和感染显像 | 某些炎症病灶定位 |
| 药物研发 | 观察药物体内分布和受体结合 |

总结：

> PET 更适合高端分子影像、肿瘤学、神经系统、心血管和精准医学场景。

## 五、核心对比表

### 1. 成像原理对比

| 项目 | γ 相机 | SPECT | PET |
|---|---|---|---|
| 全称 | Gamma Camera | Single Photon Emission Computed Tomography | Positron Emission Tomography |
| 中文名 | γ 相机 / 闪烁照相机 | 单光子发射计算机断层成像 | 正电子发射断层成像 |
| 成像方式 | 平面投影成像 | 多角度投影后断层重建 | 符合探测后断层重建 |
| 探测射线 | 单个 γ 光子 | 单个 γ 光子 | 两个 511 keV 湮灭光子 |
| 是否使用准直器 | 使用 | 使用 | 通常不使用机械准直器 |
| 图像维度 | 二维 | 三维 | 三维 |
| 是否断层 | 一般不是 | 是 | 是 |

### 2. 示踪剂与衰变类型对比

| 项目 | γ 相机 | SPECT | PET |
|---|---|---|---|
| 示踪剂类型 | 单光子 γ 发射示踪剂 | 单光子 γ 发射示踪剂 | 正电子发射示踪剂 |
| 常用核素 | Tc-99m、I-123、I-131、Tl-201、Ga-67 | Tc-99m、I-123、In-111、I-131、Tl-201 | F-18、C-11、N-13、O-15、Ga-68、Cu-64 |
| 主要衰变类型 | 同质异能跃迁、电子俘获、β⁻ 伴 γ 发射 | 同质异能跃迁、电子俘获、β⁻ 伴 γ 发射 | β⁺ 衰变 |
| 探测光子来源 | 核素直接发出的 γ 光子 | 核素直接发出的 γ 光子 | 正电子湮灭产生的 511 keV 光子 |
| 典型示踪剂 | Tc-99m-MDP、I-123、Tl-201 | Tc-99m-MIBI、Tc-99m-HMPAO、I-123-MIBG | F-18-FDG、Ga-68-PSMA、Ga-68-DOTATATE |

### 3. 成像性能对比

| 性能 | γ 相机 | SPECT | PET |
|---|---|---|---|
| 空间分辨率 | 中等 | 中等，通常低于 PET | 较高，通常优于 SPECT |
| 灵敏度 | 较低 | 较低到中等 | 较高 |
| 定量能力 | 较弱 | 中等 | 较强 |
| 解剖定位 | 较差 | SPECT/CT 可改善 | PET/CT 或 PET/MR 较好 |
| 功能信息 | 有 | 有，且有断层定位 | 很强，分子代谢信息丰富 |
| 扫描时间 | 较短 | 较长 | 中等，常较 SPECT 快 |
| 设备成本 | 较低 | 中等 | 较高 |
| 临床普及度 | 高 | 高 | 较高，但成本更高 |
| 图像重叠问题 | 明显 | 明显改善 | 明显改善 |

## 六、各自适用范围总结

| 成像方式 | 更适合的场景 |
|---|---|
| γ 相机 | 平面显像、全身筛查、动态功能检查，如骨显像、甲状腺显像、肾动态显像、肝胆显像 |
| SPECT | 需要断层定位的单光子显像，如心肌灌注、骨 SPECT、脑血流、甲状旁腺、神经内分泌肿瘤、感染灶定位 |
| PET | 高灵敏度、高定量、高分子代谢信息需求，如肿瘤分期、疗效评价、脑代谢、心肌活性、PSMA 前列腺癌显像、DOTATATE 神经内分泌肿瘤显像 |

## 七、简明记忆版

可以这样记：

```text
γ 相机：
单光子 + 平面显像 + 准直器
主要看二维分布

SPECT：
单光子 + 旋转 γ 相机 + 断层重建
主要看三维功能分布

PET：
正电子 + 511 keV 双光子符合探测 + 断层重建
主要看高灵敏度分子代谢信息
```

一句话总结：

> **γ 相机是基础平面核医学显像设备；SPECT 是 γ 相机旋转采集后的单光子断层成像；PET 是利用正电子湮灭符合探测实现的高灵敏度、高定量分子影像。**
